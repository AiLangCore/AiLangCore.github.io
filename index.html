Install

macOS and Linux

curl -fsSL https://ailang.org/install.sh | sh

Installs the current stable toolchain into ~/.ailang.

Windows PowerShell

irm https://ailang.org/install.ps1 | iex

Installs the current stable toolchain into %USERPROFILE%\.ailang.

Channels

curl -fsSL https://ailang.org/install.sh | sh -s -- --channel alpha
curl -fsSL https://ailang.org/install.sh | sh -s -- --version 0.0.1-alpha.10

The bootstrapper downloads prebuilt release artifacts, updates the local toolchain pointer, and creates shims for ailang, aivm, and aivectra.

Principles

  • Semantic authority lives inside the language.
  • The host runtime is mechanical and replaceable.
  • Determinism is mandatory.
  • Specification governs implementation.
  • Golden tests are authoritative.
